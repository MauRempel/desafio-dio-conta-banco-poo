# 💳 Desafio DIO - Conta Bancária com POO

Projeto desenvolvido como parte do desafio da Digital Innovation One (DIO) com o objetivo de aplicar os conceitos de Programação Orientada a Objetos (POO) em Java na construção de um sistema bancário simples.

## 🚀 Objetivo

Simular um sistema bancário capaz de gerenciar contas e realizar operações básicas, reforçando conceitos fundamentais de POO como:

- Abstração
- Encapsulamento
- Herança
- Polimorfismo

## 🛠️ Tecnologias Utilizadas
- Java
- Programação Orientada a Objetos (POO)
- Git & GitHub

## 📌 Funcionalidades

O sistema permite:

✅ Criar contas bancárias
✅ Realizar depósitos
✅ Realizar saques
✅ Transferências entre contas
✅ Consultar saldo
✅ Exibir extrato de transações

## 📂 Estrutura do Projeto
```bash
src/
 └── com.yourbank
     ├── model
     │    ├── account
     │    └── ...
     ├── service
     ├── exception
     └── main
```
- model → Representação das entidades (Conta, Transação, etc.)
- service → Regras de negócio (transferência, validações)
- exception → Tratamento de erros personalizados
- main → Execução da aplicação
